# 99 · Decision Research — gate-skips

When a stage is entered without its upstream output, the reason is logged here
(one file per skip) and indexed in `PROJECT.md`.

- **File naming:** `YYYY-MM-DD-<slug>.md`
- **Contents:** what gate was skipped, why, who/what decided, and the risk
  accepted.

Full guidelines: `../../.lodestar/SPEC.md` → §5.5 / `#research`.
