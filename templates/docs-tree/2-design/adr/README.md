# 2 · Design — ADR (Architecture Decision Records)

**Purpose:** record hard-to-reverse technical decisions, one per file.

- **File naming:** `ADR-NNN-<slug>.md`
- **Rule:** framework/library/architecture choices get an ADR with context,
  decision, and consequences.

Full guidelines: `../../../.lodestar/SPEC.md` → §5.2 / `#design`.
